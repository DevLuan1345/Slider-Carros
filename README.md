## 🚗 Projeto FP Selection

Este projeto consiste em uma landing page temática voltada para a apresentação de carros esportivos premium, desenvolvida com foco em design moderno, animações suaves e experiência visual interativa.

A aplicação apresenta veículos de luxo em um slider dinâmico e elegante, utilizando efeitos de transição, elementos visuais modernos e um layout responsivo que proporciona uma navegação fluida em diferentes dispositivos.

O projeto foi desenvolvido com uma identidade visual inspirada em vitrines automotivas de alto padrão, destacando os carros de forma impactante e sofisticada.

## 🚀 Funcionalidades

- 🚘 Slider interativo de carros esportivos
- 🎨 Layout moderno e visual premium
- ⚡ Transições suaves entre os veículos
- 📱 Responsividade para mobile, tablet e desktop
- 🖼️ Imagens com efeitos visuais e animações
- 🔄 Navegação dinâmica com botões laterais
- 📍 Indicadores de posição do slider
- 💻 Interface organizada e intuitiva

## 🛠️ Tecnologias Utilizadas

- HTML5 🟧
- CSS3 🟦
- JavaScript 🟨

## 🌐 Sobre o Projeto

O FP Selection foi criado com inspiração em sites modernos de concessionárias e vitrines automotivas premium, trazendo uma experiência visual sofisticada para apresentação de supercarros.

A proposta do projeto é unir design moderno, interatividade e animações para criar uma interface elegante e agradável para o usuário.

## 🎯 Objetivo

O principal objetivo do projeto é praticar o desenvolvimento front-end, explorando:

- Layouts modernos
- Responsividade para diferentes tamanhos de tela
- Criação de sliders personalizados
- Manipulação do DOM com JavaScript
- Animações e transições CSS
- Posicionamento avançado de elementos

## 💡 Diferenciais

- 🎨 Visual inspirado em vitrines automotivas premium
- 🚗 Slider totalmente personalizado sem bibliotecas externas
- 📱 Interface adaptável para qualquer dispositivo
- ⚡ Navegação dinâmica e intuitiva
- ✨ Efeitos modernos de animação e transição
- 🖥️ Estrutura limpa e organizada
